# CS157A-section1-team2
Project repository for CS157A-01 Team 2

This branch contains an Eclipse project representing the web application.

Instructions on using this project
1. The actual Eclipse project is the folder 'eVendor-eclipse'.
2. Clone this repository on your local machine.
3. In Eclipse, import ONLY the folder 'eVendor-eclipse'. Eclipse should configure the project
	accordingly.
4. Right click the project in Project Explorer -> Run As -> Run on Server
5. Follow the prompts and the web page should be displayed.
